# 📚 Spring Boot Bookstore App

A modern, full-stack library management system with a clean UI and secure user authentication.

## ✨ Features
* **User Accounts:** Secure Login and Registration system.
* **Book Management:** Admin-only access to add, edit, or delete books.
* **Personal Collection:** Users can save books to their own "My Books" list.
* **Modern Design:** Built with Bootstrap 5 and Google Fonts (Poppins) for a premium feel.
* **Persistent Storage:** Fully integrated with a MySQL database.

## 🛠️ Tech Stacks I used
* **Backend:** Java, Spring Boot, Spring Security, Spring Data JPA.
* **Frontend:** Thymeleaf, Bootstrap 5, HTML/CSS.
* **Database:** MySQL.
* **Build Tool:** Maven.
  
If You want to run it locallly:
## 🚀 How to Run Locally
1. Clone this repository.
2. Update `src/main/resources/application.properties` with your MySQL credentials.
3. Run `mvn spring-boot:run`.
4. Open `http://localhost:8080` in your browser.
